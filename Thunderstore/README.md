# THIS MOD NEEDS [PSVR2 ToolKit](https://github.com/BnuuySolutions/PSVR2Toolkit) TO WORK!

# PSVR2

Adds support for PSVR2 specific features such as adaptive triggers.

## Known Issues
- Left controller adaptive trigger does not work. This is an issue with PSVR2 Toolkit. Do not bother the developers about this issue as they have already stated that it will be fixed with the next release of PSVR2 Toolkit.

## Changelog
- 1.0.1 - Added PSVR2 specific eye tracking implementation for eye tracking mod. More convenient than the OSC/VRCFaceTracking one for PSVR2 users.
- 1.0.2 - Minor changes to the README. Removed eye tracking as an optional dependency since the mod would throw an error without it.
- 1.1.0 - Almost complete rewrite of the mod. The EyeTracking mod is no longer a hard dependency.
